## Installation
Ignore the severity vulnerabilities warnings and errors.
```
cd server-express && npm install && cd ../client-react && npm install && npm audit fix
```

## Run the Development
Terminal 1
```
cd server-express && npm run local
```
Terminal 2
```
cd client-react && npm start
```
